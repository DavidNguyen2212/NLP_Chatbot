# Chatbot với MERN stack và LLaMA 3.1:8b

**Mô tả:**

Dự án này xây dựng một chatbot thông minh sử dụng công nghệ MERN stack, kết hợp với mô hình ngôn ngữ lớn LLaMA 3.1 và nền tảng Ollama. Chatbot có khả năng hiểu và đáp ứng các câu hỏi, yêu cầu của người dùng theo dữ liệu từ trang agoda.

**Công nghệ:**

* MERN stack (MongoDB, Express.js, React, Node.js) + Typescript
* Ollama
* LLaMA 3.1

**Cài đặt:**

1. Clone repository: `git clone https://github.com/your-username/your-project.git`
2. cd NLP_Chatbot
3. Cài đặt docker desktop (Windows) hoặc docker command line (unix like)
4. docker compose up -d
5. Sau khi chạy xong lệnh này, 3 container được tạo ra và thuộc cùng network.

**Sử dụng:**

Truy cập vào địa chỉ http://localhost:5173 để bắt đầu tương tác với chatbot.
